# Checks if json is valid or not

Checks if json is valid or not, returns true or false

### Usage

```
import isJsonValid from 'is-valid-json'
```

```
const obj = {
	a: '1',
	b: '2',
	c: '3'
}

const isValid = isJsonValid(obj)
```

```
Output : 

console.log(isValid)

'true'

```
